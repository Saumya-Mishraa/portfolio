# Personal Portfolio Website

This is my personal portfolio website built to showcase my skills, projects, and contact information. The website highlights my work as a web developer and provides an overview of my technical abilities, projects, and background.

## 🚀 Features
- Responsive portfolio design
- About Me section with personal introduction
- Skills section highlighting technologies and tools
- Projects section displaying my work
- Contact form for easy communication
- Clean and modern UI design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Git & GitHub
- Responsive Web Design

## 📂 Sections
- Home
- About
- Skills
- Projects
- Contact

## 🎯 Purpose
The main goal of this portfolio is to present my development skills, share my projects, and create a professional online presence for recruiters, collaborators, and anyone interested in my work.
